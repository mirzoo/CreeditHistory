# CreeditHistory

iOS-приложение — экран кредитного рейтинга с анимированной шкалой на SwiftUI.

## Возможности

- **Анимированная шкала рейтинга** — дуга с angular-градиентом (красный → жёлтый → зелёный)
- **Точка-индикатор** — динамический цвет, соответствующий позиции на градиенте
- **Кусочно-линейный маппинг** — корректное отображение зон рейтинга:
  - 1–300 — Плохой (красная зона)
  - 301–600 — Средний (жёлтая зона)
  - 601–900 — Хороший (зелёная зона)
  - 901–999 — Отличный (ярко-зелёная зона)
- **Glow-эффект** — размытая копия шкалы создаёт эффект свечения
- **Анимация при открытии** — автоматическая демонстрация (589 → 970 → 235 → 589)
- **Перекатывание чисел** — `.contentTransition(.numericText())`
- **Кастомный easing** — `cubic-bezier(0.66, 0, 0.34, 1)`
- **Тёмная тема** — полностью кастомный дизайн
- **Карточки причин** — с иконками (Bad, Good, Normal)

## Технологии

- Swift 6
- SwiftUI
- `Animatable` протокол для плавной анимации дуги
- `AngularGradient` для градиента шкалы
- `.fullScreenCover` / `.sheet` с `.presentationDetents`

## Требования

- iOS 17.0+
- Xcode 16+
